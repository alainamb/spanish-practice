Los signos de puntuación

Introduction

Whether you're reading a technical document, navigating a file path, or working in a code editor, punctuation marks appear everywhere — and in professional and technical contexts, you'll often need to say them out loud in Spanish. This lesson covers the Spanish names for common punctuation marks, with special attention to the ones that appear most often in technology, software, and localization work.

Some marks have one widely used name; others have regional variants or informal nicknames. Where relevant, the most common alternatives are noted.

Instructions

Below you'll find a simulated work instruction for a software installation procedure. It contains many punctuation marks used as symbols in the text. Read the instruction aloud in Spanish, and when you reach a symbol, say its name in Spanish instead of reading it as a character.

For example, if you see C:\Archivos, you would read: "C, dos puntos, barra invertida, Archivos."
