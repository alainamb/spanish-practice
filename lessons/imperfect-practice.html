Imperfect Practice - Interruptions, Then and Now

Introduction

You can talk about the past in two ways in Spanish, using either the preterite or the imperfect. The preterite is for things that happened that have more definition, such as events with clear beginnings or ends. Here you're practicing the ongoing past, so you'll need to use the imperfect form.

The imperfect is used to describe ongoing or habitual actions in the past. In this activity, you'll practice forming imperfect conjugations in two common real-world contexts: describing what was happening that was interrupted, and contrasting the present with the past, including learning how to gossip about what others used to do!

These situations are designed to help you build mental scenarios for conversations. When you practice imagining yourself using the imperfect tense in these contexts, you'll develop muscle memory that makes it easier to use the tense naturally when speaking Spanish.

Instructions

This lesson features two modes for practicing imperfect conjugations.

  • In Interruptions mode, you're given a subject and a verb, and you'll need to imagine an ongoing activity that was interrupted. You can click "Show Answer" to display the correct imperfect conjugation for the first part of the sentence.Fill in words and context in the first part of the sentence, so that when combined, then whole sentence makes sense.
  • In Then and Now you're given a state of things now, and you'll imagine how things were in the past using the same subject and verb from the present tense part of the sentence. Here, clicking "Show Answer" displays the correctly conjugated imperfect tense of the verb. You can build upon that as you imagine your "then".

Remember:

  • AR verbs use the -aba endings: yo bailaba, tú bailabas, él/ella/elle bailaba, nosotros/nosotras bailábamos, ellos/ellas/elles bailaban
  • ER and IR verbs use the -ía endings: yo comía, tú comías, él/ella/elle comía, nosotros/nosotras comíamos, ellos/ellas/elles comían
  • Three irregular verbs:
    • ir → iba, ibas, iba, íbamos, iban
    • ser → era, eras, era, éramos, eran
    • ver → veía, veías, veía, veíamos, veían
