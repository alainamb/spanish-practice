Typing Accented Spanish Letters

Just the Letters

Á á É é Í í Ó ó Ñ ñ Ú ú ü ¿? ¡!

Alt codes (Windows) and commands (Mac) for typing Spanish accented letters

Character Description Alt Code (Windows) Mac
Á Uppercase A Acute Alt + 0193 Opt + e, then Shift + a
á Lowercase a acute Alt + 0225 Opt + e, then a
É Uppercase E Acute Alt + 0201 Opt + e, then Shift + e
é Lowercase e acute Alt + 0233 Opt + e, then e
Í Uppercase I Acute Alt + 0205 Opt + e, then Shift + i
í Lowercase i acute Alt + 0237 Opt + e, then i
Ó Uppercase O Acute Alt + 0211 Opt + e, then Shift + o
ó Lowercase o acute Alt + 0243 Opt + e, then o
Ñ Uppercase N Tilde Alt + 165 Opt + n, then Shift + n
ñ Lowercase n tilde Alt + 164 Opt + n, then n
Ú Uppercase U Acute Alt + 0218 Opt + e, then Shift + u
ú Lowercase u acute Alt + 0250 Opt + e, then u
ü Lowercase u umlaut Alt + 0252 Opt + u, then u
¿? Inverted question mark Alt + 168 Opt, then Shift + ?
¡! Inverted exclamation point Alt + 173 Opt, then 1
